# Lugman Khamis | Digital Portfolio

> **"Where rivers meet the desert"**

A minimalist, high-performance portfolio exploring the intersection of secure software systems, interactive game design, and community leadership. Built with the steady, purposeful flow of the Blue Nile in mind.

## 🌊 The Vision
This project is a living laboratory of my journey as a developer. Originally conceptualized in 2023, the current version was rebuilt from zero starting October 16, 2025, to transition from a standard template to a custom-coded, motion-first executive experience.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Modern Flex/Grid, Custom Properties)
- **Interactivity:** Vanilla JavaScript (Intersection Observer API, Custom Cursor logic)
- **Aesthetic:** Blue Nile Palette (Nile Navy & Egyptian Gold)
- **Deployment:** GitHub Pages
- **Domain:** [www.lugman.site](http://www.lugman.site)

## 🏗️ Core Sections
- **The Current:** My technical foundation in Python, C#, and SQL.
- **The Oasis:** A gallery of technical works including "The Sentry" (Password Manager) and "The Expedition" (Unity Jam Prototype).
- **The Bedrock:** My leadership legacy, including generating over $700 in community fundraising through FCCLA and the Ham Radio Club.

## 📜 Timeline
- **2023:** Initial concept and technical exploration.
- **Oct 2025:** Full architectural pivot to the "Blue Nile" emotional design.
- **Present:** Continuous refinement of project documentation and interactive elements.
